## 追加

- [SyncLyrics](/plugin/samples.html#synclyrics) にカスタムフォントを指定する設定を追加

## 変更

- 日本語翻訳の修正
- 選曲画面、リザルト画面の UI を統一
- 選曲画面のパフォーマンスの向上
- 一部画像の更新

## 削除

- 左端で正しく音符が表示されない不具合を削除
- 音源ファイルが存在しない場合に、再生画面で強制終了することがある不具合を削除
- デフォルトの難易度が存在しないときに曲を長押しすると強制終了する不具合を削除
